# 处理图片资源

webpack5内置了`file-loader`和`url-loader`来处理图片资源，只需简单配置即可实现对项目中图片资源的加载优化。

```javascript
const path = require('path');

module.exports = {
  ...
  module: {
    rules: [
      {
        test: /\.(png|jpe?g|gif|webp|svg)$/,
        type: 'asset',
        parser: {
          dataUrlCondition: {
            // 指定小于30kb的图片转base64
            // 优点：减少请求数量 缺点：体积可能会更大
            maxSize: 30 * 1024,
          },
        },
      },
    ],
  },
};
```

也可以通过配置来指定打包之后的图片资源输出的指定的目录：

```json
generator: {
  filename: 'static/images/[name].[ext]',
},
```
